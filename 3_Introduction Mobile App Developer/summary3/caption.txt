WEB & APPLICATION DEV 

Sudah tau belum apa itu web application developer? 
WEB APPLICATION DEVELOPER adalah pengembangan aplikasi yang didasarkan pada website, yang mana programmer memiliki tanggung jawab dalam pembuatan website dari awal hingga akhir 

WEB developer biasanya dibagi atas beberapa jenis keahlian seperti 
frontend developer ,backend developer, JavaScript developer dan full stack developer
Tips dalam memilih web application developer adalah : 
1. Pilih pengembang yang memiliki passion , karena  developer yang memiliki passion akan serius dan lebih mengutamakan kualitasnya
2. Melihat portofolio developer untuk melihat apa saja yang telah mereka kerjakan 
3. Kenali apa yang ingin anda buat , memerlukan model dan bahasa apa agar website tepat guna
